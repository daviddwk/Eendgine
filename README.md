## Instalation

### Requirements
- cmake
- SDL2
- assimp
- glad 

Glad files need to be generated for your system. See lib/glad/README.txt

### Building
```
git clone https://github.com/daviddwk/Eendgine.git
cd Eendgine
mkdir build
cd build
cmake ..
make
cd ..
```

### Executing

Just run game.out!
```
./game.out
```
